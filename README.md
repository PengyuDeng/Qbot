已有或正在开发的功能:

- [x] 今日运势查询

- [x] 今日新闻查询

- [x] 今日星座运势

- [x] 随机求签

- [x] 音乐分享（暂只支持网易云）

- [x] 发送语音（需要前往[百度开发者平台](https://ai.baidu.com/tech/speech)获取key）

- [x] 王者战力查询

- [x] 随机发送~~美女~~图片

- [x] 支持茉莉云AI（需要前往[茉莉云官网](https://mlyai.com/)获取key)（测试中）

- [x] COC鱼情查询

- [x] 群管理插件（关键字撤回、自动审核入群申请）[帮助文档](https://www.miraiqbot.xyz)（测试中）
